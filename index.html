GitHub Repo Gallery

I used Javascript and APIs (Application Programming Interface) to create a gallery of repositories. A repository allows you to store your code, your files, and each file's revision history in the GitHub cloud. When you click on a repository in the gallery, the API connects the two applications and the data is pulled, so that you can see more information about the repository.

Skills: "Javascript, APIs, Manipulate the DOM"

Tools: "Visual Studio Code, GitHub"
